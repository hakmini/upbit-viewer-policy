개인정보처리방침 (Privacy Policy)

Upbit RealTime Viewer 확장 프로그램은 어떠한 개인정보도 수집, 저장, 공유하지 않습니다.

1. 수집하는 정보

본 확장 프로그램은 사용자의 개인정보, 브라우징 정보, 계정 정보 등 어떠한 개인식별정보도 수집하지 않습니다.

본 확장 프로그램은 업비트 공개 API 데이터를 자체 서버(newchromeproject.onrender.com)를 통해 실시간으로 전달받아 UI에 표시할 뿐이며, 이 과정에서 개인정보는 전혀 관여하지 않습니다.

3. 제3자 제공

개인정보를 제3자에게 제공하거나 공유하지 않습니다.

4. 저장되는 정보

로컬 스토리지에 저장되는 데이터는 즐겨찾기 목록으로, 개인 식별과 무관한 단순 문자열이며 외부로 전송되지 않습니다.

5. 외부 서비스 사용

  • 업비트 공개 API
  • 개발자가 운영하는 데이터 중계 서버(newchromeproject.onrender.com)
